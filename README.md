# Scissors ✂️, Paper 📄, Rock 🪨
Scissors-Paper-Rock is a game based off of the popular irl game "Scissors, Paper, Rock". The player chooses one of the options and the computer chooses a random option. Depending on the options chosen, the player may win, loose or tie.

## Getting Started
This is meant to be a guide on how to use the game.

### Prerequisites
All you will need is a terminal with python installed. 

### Installation
#### Python

1. Download the latest python release 
    https://python.org/downloads
2. Install Python and you should be done

#### Scissors, Paper, Rock
1. Download the main.py file  
2. Save it where you want to

After that you should be done.

### Usage
To play the game:
1. Open up a terminal and change directories to the location you saved the main.py file
2. Once there, type __python main.py__ or __python3 main.py__ or __py main.py__ (depending on the version and OS) and press enter.
3. You will be prompted to enter the amount of rounds to play. Enter any number > 0 and press enter.
4. You will be prompted to enter what choice you want to play as. 
Options that work:
    - Full name (Scissors, Papper, Rock)
    - Single letter either upper case or lower case (S/s, P/p, R/r) 
5. It will got through each round until all rounds are completed.
6. It will output the results to the console and write the results to a results.txt file in the working directory.
